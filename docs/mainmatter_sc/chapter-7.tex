\chapter{开发路线图}
\section{路线图与融资计划}

\subsection{项目路线图}

本项目开发采用分阶段方法，于2026年第一季度启动。初始阶段活动包括融资活动、核心团队扩张、官方网站部署、白皮书发布，以及在Twitter和Discord平台建立社区。

\textbf{2026年第一季度} 启动协议架构和代币经济机制的Alpha测试，验证核心功能和经济模型参数。

\textbf{2026年第二季度} 发布供公众参与的测试网，支持在分布式基础设施上进行社区测试和反馈收集。

\textbf{2026年第三季度} 上线供公众访问的主网，同步开展初始去中心化交易所发行（IDO），标志着全面运营部署和代币分发的启动。

\subsection{融资计划}

本融资策略采用三阶段代币分配方式。2026年第一季度的初始发行分配5\%的CPT代币，目标筹集\$4M USD资金，以优惠估值向早期支持者开放。市场对比分析将本项目与成熟的去中心化计算网络进行对标，尤其是估值\$200M的Golem，其核心基础设施不足8,000个，月度使用率为\$30K。

2026年第二季度和2026年第三季度的后续融资轮次将分别以当前市场估值分配5\%的CPT代币供应量，根据已达成的里程碑和已证明的网络增长获取资金。渐进式定价反映了平台的成熟度和计算资源的扩张。
