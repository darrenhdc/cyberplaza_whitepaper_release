\chapter{愿景与使命}
%\label{chapter:title}
\begin{itemize}
\item 计算是推动人类进步的强大手段。
\item 计算推动人类进步的变革潜力，可通过一个开放、包容且民主的 Web 3 computility 平台加速实现。
\item 淘宝和拼多多作为``为人民''的企业表现出色，但并非``属于人民、由人民运营''的企业。基于 computility 开展业务的 Web3 平台可以完全成为``为人民、属于人民、由人民''的企业。
\item 理想的企业是一个拥有共同愿景的 DAO\footnote{DAO 是去中心化自治组织（Decentralized Autonomous Organization）的缩写}，而最优的 DAO 是成功的企业。
\item 我们的使命是通过运营一个以愿景为导向的 computility DAO 企业，服务于人类\footnote{``thy kind''指人类，强调我们服务全人类的承诺}。
\end{itemize}
