\chapter{願景與使命}
%\label{chapter:title}
\begin{itemize}
\item 計算科技是推動人類進步的強大工具。
\item 透過開放、包容且民主的 Web 3 計算能力平台，可加速計算科技推動人類進步的變革潛力。
\item 淘寶與拼多多作為「為了大眾」的企業表現優異，但它們並非「屬於大眾且由大眾運營」的企業。一個以計算能力營運的 Web3 平台可完全做到「為了、屬於且由大眾運營」的企業。
\item 理想的企業是具備共同願景的 DAO\footnote{DAO 代表去中心化自治組織（Decentralized Autonomous Organization）}，而最佳的 DAO 是成功的企業。
\item 我們的使命是透過以願景為導向的計算能力 DAO 企業，服務人類\footnote{「人類（thy kind）」指的是全人類，強調我們致力於服務所有的人類}。
\end{itemize}
