\chapter{願景與使命}
%\label{chapter:title}
\begin{itemize}
\item 運算技術是推動人類進步的強大工具。
\item 運算技術推動人類進步的變革潛能，可透過開放、包容且民主的 computility Web 3 平台加速發揮。
\item 淘寶與拼多多作為 ``為人民'' 的企業表現卓越，但它們並非 ``屬於人民且由人民運作'' 的企業。基於 computility 經營業務的 Web3 平台，可完全成為 ``為人民、屬於人民且由人民運作'' 的企業。
\item 理想的企業是具共識願景的 DAO\footnote{DAO 代表去中心化自主組織（Decentralized Autonomous Organization）}，而最佳的 DAO 則是成功的企業。
\item 我們的使命是透過以願景導向的 computility DAO 企業運作，服務 ``汝之族群''\footnote{``汝之族群'' 指人類全體，強調我們服務全人類的承諾}。
\end{itemize}
