\chapter{愿景与使命}
%\label{chapter:title}
\begin{itemize}
\item 计算是推动人类进步的有力手段。
\item 计算推动人类进步的变革潜力可通过一个开放、包容且民主的 Web 3 computility 平台得到加速。
\item 淘宝网与拼多多是出色的“为人民”的企业，但并非“属于人民并由人民主导”的企业。一个基于 computility 开展业务的 Web3 平台可完全实现“为人民、属于人民并由人民主导”的模式。
\item 理想的企业是拥有共同愿景的 DAO\footnote{DAO 代表去中心化自治组织}, 而最优的 DAO 是成功的企业。
\item 我们的使命是通过运营一个以愿景为驱动的 computility DAO 企业来服务全人类\footnote{“Thy kind”指人类，强调我们服务全人类的承诺}。
\end{itemize}
